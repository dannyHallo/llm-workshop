# 敬酒礼仪文化

在家庭聚会上如何敬酒、如何回应

# 场景细化

家庭聚会晚宴

# 敬酒对象细化

1. 给长辈敬酒
2. 给兄弟姐妹敬酒
3. 给小孩或晚辈饮料干杯（以鼓励或祝福的形式）
4. 给远道而来的亲戚敬酒
5. 给酒量小的家人/不喝酒的家人敬酒
6. 给新加入家庭的成员或新认识的朋友敬酒

# 行为细化

1. 长辈指出给人倒酒比较少
   - 长辈：“孩子，你这倒的酒有点少啊，是不是舍不得好酒给我们喝？”
   - 回应：“哎呀，真是抱歉，我这就给您加满，保证让您喝得开心！”

2. 家人不小心把酒弄撒了
   - 家人：“哎呀，对不起对不起，我不小心把酒弄撒了。”
   - 回应：“没事没事，酒洒了还有好酒再倒，重要的是大家聚在一起开心。”

3. 敬酒时有人说自己不会喝酒
   - 有人：“我真的不会喝酒，能不能以茶代酒啊？”
   - 回应：“当然可以，喝酒重要的是心意，不在于形式。我们以茶代酒，一样表达感情。”

4. 家庭聚会发言
   - 主持人：“感谢大家今天能聚在一起，这是我们家的幸福时刻。来，我们一起举杯，为我们的团聚干杯！”

5. 家人提议让你给大家敬酒
   - 家人：“来，今天是个好日子，你作为家里的一份子，给大家敬个酒吧！”
   - 回应：“好的，那我就借这个机会感谢大家的支持和关爱。我们一起努力，让我们的家更加和谐幸福！”

6. 家人敬酒肯定你在家庭中的付出怎么回
   - 家人：“这段时间以来，你为家里付出了很多，我们都看在眼里。今天我要向你敬个酒，表示感谢！”
   - 回应：“哎呀，这都是我应该做的。感谢大家的认可和支持，我会继续努力为家庭付出的。
